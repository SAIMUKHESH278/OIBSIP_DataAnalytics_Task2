# Google Play Store Data Analysis

## Objective
The objective of this project is to analyze Google Play Store applications to understand app market trends, user preferences, ratings, installs, pricing, and user sentiments.

## Dataset
The project uses Google Play Store app data and user review datasets containing information about applications, ratings, reviews, categories, installs, prices, and sentiments.

## Steps Performed
- Loaded Google Play Store datasets
- Cleaned and transformed data types
- Removed duplicate records and handled missing values
- Analyzed app categories and their distributions
- Studied ratings, reviews, installs, and pricing trends
- Compared free and paid applications
- Performed user sentiment analysis based on reviews
- Created visualizations to identify market patterns and insights

## Tools and Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code

## Outcome
The analysis provided valuable insights into the Android application market, including popular categories, user behavior, app ratings, and sentiment trends, helping to understand factors influencing app success.
